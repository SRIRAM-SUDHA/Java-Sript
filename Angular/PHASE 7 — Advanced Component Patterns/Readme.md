# PHASE 7 — Advanced Component Patterns

## Goal
Master advanced composition patterns (projection, queries) and scalable communication patterns.

## Topics
1. **Content Projection**
   - `ng-content`
   - Multi-slot projection
   - React children vs Angular projection
2. **View & Content Queries**
   - `@ViewChild`, `@ContentChild`
   - Lifecycle timing
   - DOM access patterns
3. **Component Communication Patterns**
   - Input/Output vs shared services
   - Store-like patterns
   - Avoiding prop-drilling equivalents

## Hands-on (Do This)
- Build a reusable “Card” or “Modal” component using projection (including multi-slot).
- Use queries to integrate with a child component API (not for random DOM poking).
- Refactor one area to reduce “prop-drilling” by introducing a facade/store service.

## Exit Criteria
- You can design flexible, reusable UI components without hacks.
